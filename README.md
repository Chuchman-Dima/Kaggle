# Kaggle Solutions Repository
Цей репозиторій створений для зберігання моїх рішень задач із платформи Kaggle. Тут я практикую навички Data Science, Machine Learning та аналізу даних.

## 🎯 Мета проєкту
- Закріплення теоретичних знань на реальних датасетах.
- Вивчення різних алгоритмів (Linear Regression, Random Forest, XGBoost тощо).
- Відпрацювання повного циклу роботи з даними: від очищення (EDA) до валідації моделі.

## 🛠 Технологічний стек
- Мова: Python
- Бібліотеки: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- Середовище: Jupyter Notebook / PyCharm
